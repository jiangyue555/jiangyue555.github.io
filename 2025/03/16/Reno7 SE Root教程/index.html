适用于OPPO联发科1200以下的Root教程

2025/3/16 Root BL

此教程不限版本系统 只要是oppo realme都行

需要的所有文件都在资源站alist.jiangyue.cyou

本次演示设备-OPPO Reno7 SE 安卓13

图片我就用链接代替了 服务器速度太慢

下面教程开始：

首先先进行刷机必备环节安装驱动，先安装联发科必备驱动

如图

https://img.picui.cn/free/2025/03/16/67d629ccb9bcf.jpg

如果你是安卓11以下使用此教程需要额外安装下面驱动

https://img.picui.cn/free/2025/03/16/67d62a38a8cc0.png

下面去某🐟租一个unlocktool账号 价格2元左右

有的网络可以直接打开 打不开的话请使用魔法上网

https://img.picui.cn/free/2025/03/16/67d62b0f7db8b.png

输入账号密码后直接点同意

https://img.picui.cn/free/2025/03/16/67d62b521e50b.png

然后不出意外的话登录成功

进去主界面后点上方的mtk区

https://img.picui.cn/free/2025/03/16/67d62bbc4d69f.png

点击Preloader Auth 选择OPPO Realme V3

https://img.picui.cn/free/2025/03/16/67d62c6caa607.png

https://img.picui.cn/free/2025/03/16/67d62c86e28fc.png

好了之后点Boot Devices

等待出现如下提示https://img.picui.cn/free/2025/03/16/67d62cae73ac7.png

将手机使用adb命令行输入

adb reboot fastboot

重启到fastbootd模式 选择简体中文 连续点击下方的版本 弹出是否进入下载模式点确定

https://img.picui.cn/free/2025/03/16/67d62e51e7ab2.jpg

如果没有弹出这个界面 或者进不去fastbootd 你可以同时按住三个键 等到工具显示识别的一瞬间松手

https://img.picui.cn/free/2025/03/16/67d62fe4756b1.png

然后点击Unlock BL 解锁bl

https://img.picui.cn/free/2025/03/16/67d6303315b13.png

在上方分区列表找到Boot分区 选中后右键点Read Boot 如果是ab/vab分区就把boot_ab都提取一遍 提取到你能找到的位置

https://img.picui.cn/free/2025/03/16/67d630a88c4c0.png

然后点Reboot右面的展开 点To Recovery以重启到恢复模式

https://img.picui.cn/free/2025/03/16/67d630ec467e4.png

此时手机可能会多重启几次 不要慌张 等待就行 进去恢复模式后点简体中文/格式化数据分区，然后等待开机进入系统

激活后安装上面具或ap 将提取好的boot/boot_ab复制到手机能找到的位置 使用面具修补 修补好的镜像在download下 将其复制到电脑

再次使用adb命令行 输入以下命令:

adb reboot fastboot

fastboot flash boot （修补好的文件路径）.img

fastboot reboot

然后开机 即可

提取完并过完签名的boot已上传资源站 欢迎下载
